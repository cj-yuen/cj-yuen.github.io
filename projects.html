MineSweeper Game

For my final homework project for my CIS 1200 course, I undertook the challenge of implementing the classic Minesweeper game. The Minesweeper project showcases my proficiency in Java programming and object-oriented design. The game features a graphical user interface (GUI) built using Java Swing, providing a visually appealing and user-friendly experience.

Learn More ▼

Object-Oriented Design: The Minesweeper implementation follows a modular and object-oriented design, with classes representing various components such as the game board, tiles, and the game itself.
File I/O: I integrated file input/output operations to save and load game states enabling storage of game data, which allows users to resume their games at a later time.
Iterator implementation: The project includes a custom iterator, MineSweeperIterator, demonstrating my ability to work with Java's built-in features to create a flexible and efficient solution.
Graphics Interface (GUI): he game features an intuitive GUI built with Java Swing.
